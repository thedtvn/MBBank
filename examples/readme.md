# Examples

This directory contains example scripts demonstrating how to use the `mbbank` library.

## Available Examples

-   **[transfer_money.py](transfer_money.py)**: Demonstrates how to authenticate, list accounts, select a destination bank, and perform a money transfer.
-   **[transaction_history.py](transaction_history.py)**: Demonstrates how to retrieve and display transaction history for the last 30 days.
-   **[card_list.py](card_list.py)**: Demonstrates how to retrieve and display the list of cards.
-   **[saving_list.py](saving_list.py)**: Demonstrates how to retrieve and display the list of saving accounts.

## Usage

To run an example, simply execute the python script:

```bash
cd examples
python <name file>
```

You will be prompted to enter your MBBank username and password.
