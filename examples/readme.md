# Examples

> [!WARNING]
> This example performs real money transfers.
>
> Use with caution and ensure you have the correct credentials and destination account information.
>
> Transfer feature requires MBBank APP on your phone to scan OTP code and confirm the transfer.

This directory contains example scripts demonstrating how to use the `mbbank` library.

## Available Examples

- **[transfer_money.py](transfer_money.py)**: Demonstrates how to authenticate, list accounts, select a destination
  bank, and perform a money transfer.
- **[bulk_transfer.py](bulk_transfer.py)**: Demonstrates how to perform multiple transfers in a single batch using the
  `makeBulkTransfer` method.
- **[transaction_history.py](transaction_history.py)**: Demonstrates how to retrieve and display transaction history for
  the last 30 days.
- **[card_list.py](card_list.py)**: Demonstrates how to retrieve and display the list of cards.
- **[saving_list.py](saving_list.py)**: Demonstrates how to retrieve and display the list of saving accounts.
