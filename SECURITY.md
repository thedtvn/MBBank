# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| Latest  | :white_check_mark: |

Only the latest release receives security updates. Please upgrade before reporting.

## Reporting a Vulnerability

Use
the [Security Vulnerability Report](https://github.com/thedtvn/MBBank/issues/new?template=security_vulnerability.yml)
issue.

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Do not report if the following applies:

- AI generated security reports
- Security vulnerabilities in dependencies (report to the dependency instead)
